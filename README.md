### Installation

- Clone this repo:

```sh
git clone https://github.com/hebersolano/property-pulse.git
```

- Install dependencies:

```sh
npm install
```

- Create a dev.local file with this variables:

```env

```

- Build command:

```sh
npm run build
```

- Live server:

```sh
npm run dev
```
